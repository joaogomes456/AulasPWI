Tags Html


Doctype: Serve para avisar o navegador que versçã esta sendo Html esta sendo usada
Title: Serve para dar nome a sua pagina que ira aparecer na janela,guia ou aba.
Tag de Comentario: Serve para deixar um aviso ou comentario para você lembra porque colocou aquela tag
Footer: Serve para criar o rodape da pagina.
Article: Serve para criar um artigo para a sua pagina e organizalas separadamente.
Span: Servem para apresentar pequenas informaçoes como legendas de uma foto.
Hr: Essa tag cria uma linha otizontal em sua pagina.
A: Esta tag serve para armazenar um link que redireciona você para outra pagina.
Nav: Ela cria um conteudo para navegação, é muito usada para criar menus.
Img: Ela insere um imagem em sua pagina.

Explicando os atributos das tags ID,STYLE e CLASS


Id Define um nome unico a cada documento, el serve para identificar algum elemento ao navegar.
Style: Aplica um estilo a ser usado em algum documento ou um serto texto.
Class: Cria uma lista com classes de elementos pré-determinados.
